Introduction

In this adventure game that runs on a command-line interface, you take on the role of a knight who embarks on a quest to navigate through a maze and reach the end. Along the way, you'll need to collect various objects and engage in battles with formidable monsters in order to progress. With each new game, the layout of the maze can be different based on your selected map, providing a unique and exciting challenge.

The maps are created by parsing xml files and creating each element as an object with the appropriate properties set. By utilizing object-oriented programming, this game can adopt complex elements and characters with unique actions. My version of Rogue is written in Java and uses multithreading for running multiples processes in the game.

Game Controls:
  • 'H': Left
  • 'L': Right
  • 'K': Up
  • 'J': Down
  • 'P': Pick up item
  • 'D' [integer]: Drop item [integer] from the pack
  • 'I': Inventory
  • 'C': Change out of armor (placed back in pack)
  • 'E' [Y|y]: End the game
  • '?': Help menu showing different commands
  • 'H' [command]: Tells you more about the command
  • 'R' [integer]: Read item [integer] from pack (used for scrolls)
  • 'T' [integer]: Take out weapon [integer]
  • 'W' [integer]: Wear armor [integer]
Character Meanings:
  • '@' = Player
  • 'X' = Wall
  • '.' = Floor
  • '+' = Door
  • 'T' = Troll
  • 'S' = Snake
  • 'H' = Hobgoblin
  • ']' = Armor
  • ')' = Sword
  • '?' = Scroll
